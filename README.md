# 🤖 AI Smart Recruitment Portal

An AI-powered recruitment platform that streamlines the hiring process by connecting job seekers, recruiters, and administrators through an intelligent and user-friendly web application.

---

## 📌 Project Overview

The **AI Smart Recruitment Portal** is a modern recruitment system designed to simplify the hiring process. It enables candidates to search and apply for jobs, recruiters to manage job postings and applications, and administrators to oversee the complete recruitment workflow.

The project is built using modern web technologies with an intuitive interface and scalable architecture.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/lasya43/AI-Smart-Recruitment-Portal_.git
```

Go to the project folder

```bash
cd AI-Smart-Recruitment-Portal_
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```text
http://localhost:5173
```
